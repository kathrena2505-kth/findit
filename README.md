# 📱 FindIt - Lost & Found App

FindIt is a mobile application built using Flutter that helps users report, search, and recover lost items easily and efficiently.

---

## 🚀 Features

🔍 Search Lost Items  
- Browse items reported by others  
- Filter and find items easily  

📤 Report Lost Items  
- Add item details (name, description, location)  
- Upload images  

📦 View Found Items  
- Check items reported as found  
- Connect with users  

🔥 Firebase Integration  
- Real-time database  
- Secure backend  

🎨 Clean UI  
- Simple and user-friendly design  

---

## 🛠 Tech Stack

- Flutter  
- Dart  
- Firebase
